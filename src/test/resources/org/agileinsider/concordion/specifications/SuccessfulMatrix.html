Successful Matrix

Example

When the Matrix extension is installed that adds the matrix command in the http://www.agileinsider.org/concordion/plus namespace. This command will highlight passing matrix fixtures.

Running a specification containing: 

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
        <table cp:matrix="multiply(#row,#column)">
            <thead>
            <tr>
                <th>times</th>
                <th>7</th>
                <th>12</th>
            </tr>
            </thead>
            <tr>
                <td>3</td>
                <td>21</td>
                <td>36</td>
            </tr>
        </table>
</div>

Should produce:

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
        <table cp:matrix="multiply(#row,#column)">
            <thead>
            <tr>
                <th>times</th>
                <th>7</th>
                <th>12</th>
            </tr>
            </thead>
            <tr>
                <td>3</td>
                <td class="success">21</td>
                <td class="success">36</td>
            </tr>
        </table>
</div>
