Successful CSV Matrix

Example

When the CSV Matrix extension is installed that adds the csv-matrix command in the http://www.agileinsider.org/concordion/plus namespace. This command will highlight passing CSV matrix fixtures.

With a csv file containing:

times,7,12
3,21,36
9,63,108

And a fixture which has the method 

int multiply(int row, int column)

Running a specification containing: 

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <a href="SuccessfulCsvMatrixExample.csv" cp:csv-matrix="multiply(#row, #column)">csvfile.csv</a>
</div>

Should produce html containing:

  • <td>3</td><td class="success">21</td><td class="success">36</td>
  • <td>9</td><td class="success">63</td><td class="success">108</td>
